Instruções de Instalações

CodeIgniter é instalado em quatro etapas:

  1. Descompacte o pacote.
  2. Suba os diretórios e arquivos para o servidor. Normalmente o arquivo index.php estará em sua raiz.
  3. Abra o application/config/config.php arquivo com um editor de texto e altere a sua URL base. Se você pretende usar criptografia ou sessões, definir a sua chave de criptografia.
  4. Se você pretende usar um banco de dados, abra o aplicativo application/config/database.php arquivo com um editor de texto e definir suas configurações de banco de dados.

Se desejar aumentar a segurança, escondendo a localização dos arquivos do CodeIgniter, pode renomear o system e application pastas para algo mais privado. Se você renomeá-los, você deve abrir o seu principal index.php arquivo e defina a $system_folder e $application_folder variáveis no topo do arquivo com o novo nome que você escolheu.

Para a melhor segurança, tanto o system e qualquer application pastas devem ser colocadas acima da raiz para que elas não seja diretamente acessadas através de um navegador web. Por padrão, o .htaccess são incluídos em cada pasta para ajudar a impedir o acesso direto, mas o melhor é removê-los do acesso do público total no caso de alterações na configuração do servidor web ou não cumprir o .htaccess.

após movê-los, abra o seu index.php principal, e defina os $system_folder e $application_folder variáveis, de preferência com um caminho completo, ex: '/www/MyUser/system'.

É isso aí!

Se você é novo no Code Igniter, por favor leia a Introdução do Guia do Usuário para começar a aprender construir aplicações dinâmicas em PHP. Divirta-se!
